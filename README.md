## 스터디 규칙
- 금요일마다 공통 문제 4개 게시
- 주에 공통문제 최소 2개 + 개인 문제 최소 2개 풀기
- 금요일 기준 일주일에 3문제 이하로 풀면 2000원 벌금
- 주 1회 대면 스터디
    - 금요일 17시
    - 일주일동안 했던 공부 내용 공유
- 푼 문제는 repo에 각자 파일 생성하여 올리기
 
## Package Convention
```
{주차} / {영어 이름} / {푼 문제 파일}
```
 ex) week1 / youbin / 1920.cpp
